常見問題

關於 ArcSign 的所有疑問，都能在這裡找到答案

基本問題

ArcSign 是一個免費的 USB 冷錢包應用程式。它將你的私鑰存儲在 USB 隨身碟上，而不是電腦硬碟中，大幅降低被惡意軟體竊取的風險。

ArcSign 支援多條區塊鏈（Ethereum、BSC、Polygon 等），並透過 WalletConnect 讓你可以安全地連接到各種 dApp。

最大的差別在於私鑰存儲位置：

• MetaMask 把私鑰存在電腦硬碟，任何能讀取檔案的惡意軟體都可能竊取

• ArcSign 把私鑰存在 USB 隨身碟，拔掉就是冷錢包，攻擊者無法存取不存在的檔案

不是，ArcSign 是「強化版軟體錢包」。核心差異是私鑰存在 USB 而不是電腦硬碟，拔掉 USB 後駭客無法存取。但簽名仍在電腦完成，無法達到硬體錢包的完全隔離。

定位：比 MetaMask 更安全，比硬體錢包更便宜（免費）。大額長期儲存建議搭配硬體錢包使用。

目前支援 macOS（Intel 和 Apple Silicon）。Windows 和 Linux 版本即將推出。

安全相關

ArcSign 採用業界頂級的加密標準保護你的私鑰，安全性不輸專業級產品。

最重要的是：私鑰永遠不會離開你的 USB 隨身碟，也不會傳送到任何伺服器。拔掉 USB，駭客就無法存取你的私鑰。

ArcSign 提供兩種備份方式：

• 助記詞備份：將 12 或 24 個英文單詞抄寫在紙上，多份異地保存。即使 USB 損壞，你可以用助記詞在 ArcSign 或任何支援 BIP39 的錢包恢復資產。

• 加密檔案備份：直接複製 USB 內的錢包資料夾到另一個 USB 或安全位置。檔案已加密，只有知道密碼的人才能使用。這種方式更方便，且不用擔心助記詞外洩。

ArcSign 不儲存你的密碼，所以無法幫你重設密碼。但如果你有備份助記詞，可以用助記詞重新匯入錢包並設定新密碼。

這就是為什麼我們反覆強調備份助記詞的重要性。

有些詐騙者會空投假代幣到你的錢包，試圖誘騙你點擊惡意網站。ArcSign 會自動隱藏這些可疑代幣，只顯示已知的合法代幣。

你隨時可以選擇查看被過濾的代幣。

功能相關

WalletConnect 是一個開放協議，讓你的錢包可以安全連接到各種 dApp（如 Uniswap、OpenSea、PancakeSwap）。

使用方式很簡單：在 dApp 選擇「WalletConnect」連接方式，然後在 ArcSign 中貼上連接連結即可。所有交易都會在 ArcSign 中確認，確保你的私鑰安全。

ArcSign 透過 WalletConnect 支援所有主流 dApp，包括但不限於：

• DEX：Uniswap、PancakeSwap、SushiSwap

• NFT：OpenSea、Blur、LooksRare

• DeFi：Aave、Compound、Lido

只要 dApp 支援 WalletConnect 連接方式，你就可以使用 ArcSign。

目前完全免費！你只需支付區塊鏈網路的 Gas Fee。

未來我們將推出「智能路由」功能，整合更多聚合器自動搜尋最佳交易路徑。該功能會收取固定的低手續費（非百分比），比其他軟體錢包收取 0.5%~1% 的交易手續費便宜許多。

現有的基本 Swap 功能將維持免費。

目前支援的區塊鏈包括：

• Ethereum (ETH)

• BNB Smart Chain (BSC)

• Polygon (MATIC)

• Arbitrum One

• Optimism

• Base

我們會持續添加更多鏈的支援。

會員與付費

免費版可以使用 1 個錢包，其他所有功能都完整開放：

• USB 離線存儲

• 多鏈支援

• Token Swap

• WalletConnect

• 交易歷史

對於大多數用戶來說，免費版已經足夠。

Pro 會員資格以 BSC 鏈上的 NFT 形式存在。購買後，NFT 會鑄造到你的錢包地址。

這代表你的會員資格是真正屬於你的數位資產，可以轉讓、出售或保留。每個 NFT 增加 3 個錢包配額（1 NFT = 4 個錢包）。

在 ArcSign 應用程式中：

1. 進入「設定」→「會員資格」

2. 選擇要購買的 NFT 數量

3. 使用 USDT (BSC) 完成支付

4. NFT 會自動鑄造到你的錢包
