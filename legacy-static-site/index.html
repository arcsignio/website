ArcSign 安全設計

USB 隔離 = 拔掉就安全

把硬體錢包的核心概念帶給每個人，零額外成本

1

USB-only Storage

私鑰不在電腦硬碟

  • 私鑰只存在 USB 隨身碟
  • 拔掉 USB = 駭客找不到私鑰
  • 惡意軟體無法竊取不存在的檔案
2

Memory Hardening

記憶體層級保護

  • mlock 防止私鑰被換出到 swap
  • SecureZero 使用後立即清除
  • Split key 分片儲存，提高竊取難度
3

密碼不落地

每次操作需重新輸入

  • 密碼永不儲存在任何地方
  • Session token 時效控制
  • 閒置自動鎖定
4

加密備份

比助記詞更安全的備份方式

  • 複製錢包資料夾即可備份
  • 可存雲端，無外洩風險
  • 即使官方也無法解密

誠實說明：ArcSign 是「強化版軟體錢包」，不是硬體錢包。 簽名仍在電腦完成，但私鑰存在 USB 大幅降低被竊風險。 大額長期儲存建議搭配硬體錢包。

常見問題

你可能想知道的

最大的差別在於私鑰存儲位置。MetaMask 把私鑰存在電腦硬碟，任何能讀取檔案的惡意軟體都可能竊取。ArcSign 把私鑰存在 USB 隨身碟，拔掉就是冷錢包，攻擊者無法存取不存在的檔案。

深入比較：ArcSign vs MetaMask →

ArcSign 提供兩種備份方式：

加密檔案備份（推薦）：直接複製 USB 內的錢包資料夾到另一個 USB 或雲端。檔案已加密，只有知道密碼的人才能使用，不用擔心外洩。

助記詞備份：將 12 或 24 個英文單詞寫在紙上保存。可在任何支援 BIP39 的錢包恢復資產。

不是，ArcSign 是「強化版軟體錢包」。核心差異是私鑰存在 USB 而不是電腦硬碟，拔掉 USB 後駭客無法存取。但簽名仍在電腦完成，無法達到硬體錢包的完全隔離。

定位：比 MetaMask 更安全，比硬體錢包更便宜（免費）。大額長期儲存建議搭配硬體錢包使用。

ArcSign 採用業界頂級的加密標準保護你的私鑰，安全性不輸專業級產品。私鑰永遠不會離開你的 USB 隨身碟，也不會傳送到任何伺服器。

WalletConnect 是一個開放協議，讓你的錢包可以安全連接到各種 dApp（如 Uniswap、OpenSea、PancakeSwap）。使用方式很簡單：在 dApp 選擇「WalletConnect」連接方式，然後在 ArcSign 中貼上連接連結即可。所有交易都會在 ArcSign 中確認，確保你的私鑰安全。

有些詐騙者會空投假代幣到你的錢包，試圖誘騙你點擊惡意網站。ArcSign 會自動隱藏這些可疑代幣，只顯示已知的合法代幣。你隨時可以選擇查看被過濾的代幣。

ArcSign 的 Swap 功能完全免費，我們不收取任何額外手續費。你只需支付區塊鏈網路的 Gas Fee。我們整合了 DEX 聚合器，幫你找到最佳兌換價格。

支援 macOS（Apple Silicon）、Windows（x64）、Linux（x64），三平台皆可下載使用。

ArcSign 透過 WalletConnect 支援所有主流 dApp，包括但不限於：Uniswap、PancakeSwap、SushiSwap、OpenSea、Aave、Compound 等。只要 dApp 支援 WalletConnect 連接方式，你就可以使用 ArcSign。

Pro 會員資格以 BSC 鏈上的 NFT 形式存在。購買後，NFT 會鑄造到你的錢包地址。這代表你的會員資格是真正屬於你的數位資產，可以轉讓、出售或保留。每個 NFT 增加 3 個錢包配額（1 NFT = 4 個錢包）。

OR

手動安裝

  1. 1. 點擊上方按鈕下載對應平台安裝檔
  2. 2. 安裝到 USB 隨身碟
  3. 3. macOS 用戶需在終端機執行（將 USB_NAME 替換為你的隨身碟名稱）：

為何需要這個步驟？

macOS 會對從網路下載的應用程式加上隔離標記。這個指令只是移除這個標記，不會修改應用程式本身。
