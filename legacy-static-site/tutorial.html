使用教學

從安裝到日常使用，完整的 ArcSign 使用指南

1 安裝 ArcSign 到 USB

下載與安裝（隨插即用）

  • 準備一個專用的 USB 隨身碟（建議 16GB 以上）
  • 前往 下載頁面 下載適合你系統的版本
  • 將下載的 ArcSign 應用程式直接拷貝到 USB 隨身碟
  • 之後只要插入 USB，雙擊 ArcSign 即可使用

💡 隨插即用

把 ArcSign 和錢包資料都放在 USB 內，你可以在任何電腦上（Windows、macOS、Linux）插入 USB 直接使用，真正實現「帶著錢包走」。

⚠️ 首次開啟

首次在新電腦上開啟時，macOS 可能會提示「無法驗證開發者」，請到「系統設定 → 隱私權與安全性」點擊「強制打開」。

2 首次設定

啟動 ArcSign

  • 從 USB 隨身碟雙擊開啟 ArcSign
  • 應用程式會自動偵測並使用 USB 作為錢包存儲位置
  • 設定一個強密碼（至少 12 個字元，需包含大小寫英文和數字）

⚠️ 重要

請務必記住你的密碼！密碼不會儲存在任何地方，忘記密碼將無法恢復錢包。

設定區塊鏈節點 (RPC Provider)

ArcSign 需要透過 RPC Provider 與區塊鏈互動。目前需要手動設定 API Key：

  • 前往「設定 → Provider 設定」
  • 選擇 Provider 類型並輸入你的 API Key

💡 支援的 Provider

Alchemy — Ethereum、Polygon、Arbitrum、Optimism、Base
NodeReal — BNB Smart Chain (BSC)

⚠️ 如何取得 API Key

前往 Alchemy 以及 NodeReal 免費註冊帳號， 建立專案後即可取得 API Key。免費方案足夠一般使用。

🚀 即將推出：內建節點

未來版本將內建預設的 RPC 節點，讓你無需手動設定即可直接使用。敬請期待！

3 建立或匯入錢包

建立新錢包

  • 點擊「建立新錢包」
  • 為錢包取一個名稱（選填，例如：主要錢包）
  • 設定密碼（至少 12 個字元，需包含大小寫英文和數字）
  • 選擇助記詞長度：12 個字或24 個字（預設 24 字，安全性較高）
  • 進階選項：可設定 BIP39 Passphrase（選填，相當於第 25 個字，強烈建議設定）
  • 系統產生助記詞後，將其抄寫在紙上妥善保管
  • 依序輸入助記詞進行驗證，確認備份正確

💡 關於助記詞長度

24 個字提供更高的安全性（256 位元熵），適合大額資產。12 個字（128 位元熵）也足夠安全，且更容易備份。

🔐

為什麼強烈建議設定 BIP39 Passphrase？

你的「第 25 個字」，提供雙重保險

🛡️
防止助記詞外洩

即使有人取得你的 24 個字，沒有 Passphrase 也無法存取資產

👻
隱藏錢包

同一組助記詞搭配不同 Passphrase 會產生完全不同的地址

🚨
對抗物理威脅

被迫交出助記詞時，可只交出沒有 Passphrase 的空錢包

💾

備份你的錢包（推薦）

比傳統助記詞備份更安全的方式

直接複製 USB 內的錢包資料夾到另一個 USB 或安全位置即可。

🔒 檔案已加密，只有知道密碼的人才能使用
👁️ 不用擔心助記詞被他人看到或拍照
☁️ 可安全儲存在雲端（Google Drive、iCloud）
🔐 即使是我們也無法在沒有密碼的情況下解密

只要記得密碼，就能在任何電腦上用 ArcSign 恢復錢包。

💡 傳統助記詞備份

如果你偏好傳統方式，也可以將 12/24 個助記詞手寫在紙上保管。但請注意：任何看到助記詞的人都能存取你的資產，請確保紙本備份的安全。

匯入現有錢包

  • 點擊「匯入錢包」
  • 輸入你的 12 或 24 個助記詞
  • 如有設定 BIP39 Passphrase，請一併輸入
  • 選擇要匯入的地址（可選擇多個）
  • 設定錢包密碼

4 日常使用

查看資產

  • 開啟 ArcSign，選擇你的錢包
  • 輸入密碼解鎖
  • 在主畫面可以看到所有鏈的資產總覽
  • 點擊任一鏈可以查看該鏈的代幣詳情

發送代幣

  • 選擇要發送的代幣
  • 點擊「Send」按鈕
  • 輸入收款地址和金額
  • 確認交易詳情和 Gas 費用
  • 輸入密碼簽署交易

💡 提示

首次發送前，建議先用小額測試，確認地址正確。

兌換代幣 (Swap)

  • 點擊「Swap」功能
  • 選擇要兌換的代幣對
  • 輸入兌換金額
  • 查看預估收到的數量和滑點
  • 確認並簽署交易

5 連接 dApp (WalletConnect)

使用 WalletConnect 連接 dApp

  • 在瀏覽器開啟你想使用的 dApp（如 Uniswap、OpenSea）
  • 點擊 dApp 的「Connect Wallet」按鈕
  • 選擇「WalletConnect」選項
  • 複製顯示的連接連結或 QR Code 內容
  • 在 ArcSign 中點擊「WalletConnect」
  • 貼上連接連結
  • 確認連接請求

⚠️ 連接前請務必確認

只連接你信任的 dApp！惡意網站可能偽裝成知名 dApp，誘騙你簽署惡意交易。 連接前請確認網址正確，例如 Uniswap 是 app.uniswap.org，不是 uniswap-app.com。

💡 簽名前仔細檢查

連接後，dApp 發起的所有交易都會在 ArcSign 中彈出確認視窗。請仔細檢查交易內容、金額、合約地址後再決定是否簽署。不確定就拒絕！

6 安全建議

🛡️

ArcSign 的多層防護

專為「隨插即用、任何電腦都能用」設計

🔐
私鑰隔離

私鑰只存在 USB，電腦硬碟上無任何痕跡

🧠
記憶體保護

mlock + SecureZero，防止私鑰被換出或殘留

🔑
密碼不落地

密碼永不儲存，每次操作重新輸入

⏱️
Session 時效控制

閒置自動鎖定，降低被竊取風險

99%+ 攻擊阻擋率

即使電腦上有惡意軟體，攻擊者也需要在你操作的那幾分鐘內同時突破多層防護，難度極高。

日常安全習慣

  • 不使用時，拔掉 USB 隨身碟並妥善保管
  • 定期備份助記詞（建議多份異地保管）
  • 在陌生電腦上使用時，完成操作後立即拔掉 USB
  • 仔細檢查每一筆交易的細節後再簽署
  • 不要點擊來路不明的連結或下載可疑檔案
  • 大額資產建議搭配硬體錢包使用

如果發現異常

  • 立即拔掉 USB 並斷開網路
  • 使用另一個安全設備將資產轉移到新地址
  • 檢查是否有未授權的代幣授權（Approval）
  • 考慮建立新錢包，將資產轉移過去
